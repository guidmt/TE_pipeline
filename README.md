# TE_pipeline
 
# How to run:
1) Install bpipe
2) Configure the parameters of *.bpipe files 
3) Run:

bpipe  step1_te_pipeline.bpipe

# Dependencies:
- bpipe (last version, http://docs.bpipe.org)
- blastn (last version)
- seqkit (last version, https://github.com/shenwei356/seqkit)
- R: readr, data.table
- Perl
- repeatmasker
- abyssis
